# 智慧食堂品牌标志

## 核心概念

主标志名为“智食碗”。下方的碗代表真实可购买的一餐；上方三根圆头柱同时表达热气、营养数据和个性化推荐；暖金圆点代表被服务的学生与其健康目标。

## 标准文件

- `logo-mark.svg`：透明背景主标志，优先用于白色或浅色背景。
- `logo-mark-reversed.svg`：反白标志，用于深绿色或深色背景。
- `logo-lockup.svg`：主标志与“智慧食堂 / SMART CANTEEN”横向组合。
- `app-icon.svg`：带绿色圆角底的应用图标版本。
- 同名 `.png` 文件：`512px` 独立标志、应用图标与高分辨率横向组合导出。
- `public/favicon.svg`：网页浏览器图标。

源文件位于 `src/assets/brand/`，小程序运行时副本位于 `miniapp/src/static/brand/`。

## 品牌色

| 用途 | 色值 |
| --- | --- |
| 品牌绿 | `#167A5B` |
| 深绿色 | `#0C6248` |
| 暖金色 | `#F4B84E` |
| 主文字 | `#18251F` |
| 辅助文字 | `#64705F` |

## 使用规则

- 数字端独立标志不小于 `24px`，推荐不小于 `32px`；印刷不小于 `7mm`。
- 四周安全区至少为中间数据柱宽度的两倍。
- 深色背景必须使用反白版；应用商店头像、小程序头像使用 `app-icon.svg`。
- 不拉伸、不旋转、不改变图形比例，不给母版增加渐变、描边或投影。
- “Enterprise MVP”是开发阶段说明，不属于正式品牌名称，不写入标志。
- 正式注册或商业发布前，应进行图形商标与应用商店近似图标检索。
