## 2026-07-12 租户隔离与空数据修复

**背景**：线上评价中心、数据中心、校园环境、智能体实验室、AI 配置（含租户管理）页面存在三类问题：
1. **空数据**：环境页只显示标题无表单、租户列表引用不存在的 store 属性
2. **串租户**：AI 设置全局 fallback 导致租户间共享 API key、RAG 搜索无 tenant 过滤、Agent 工具全局单例闭包捕获旧租户上下文
3. **复制数据**：seed 全表 COUNT=0 判断无 tenant 维度，已有其他租户数据时跳过默认环境初始化
